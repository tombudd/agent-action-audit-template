# Public Claim Boundary

Allowed public wording:

- "local demo"
- "deterministic example classifier"
- "receipt-backed local examples"
- "preview-versus-execution teaching artifact"

Disallowed public wording:

- "validated by outside parties"
- "approved for release"
- "ready for live operation"
- "certified"
- "guaranteed"
- "proves alignment"

Local tests and receipts are evidence of this repository's demo behavior. They are not release approval.
